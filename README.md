# Muro de Códigos Comunidad - Seedance

Este proyecto es un muro de la comunidad donde los usuarios pueden compartir códigos de referencia.

## Tecnologías

- React
- Vite
- Tailwind CSS
- Shadcn UI
- Supabase

## Setup

1.  Clona el repositorio.
2.  Instala las dependencias: `npm install`
3.  Crea un archivo `.env` o `.env.local` con las credenciales de Supabase:
    ```
    VITE_SUPABASE_URL=your_project_url
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```
4.  Inicia el servidor de desarrollo: `npm run dev`
