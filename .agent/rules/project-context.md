# Contexto del Proyecto: Comunidad de Códigos

Estás construyendo una plataforma web para una comunidad de desarrolladores.

## 1. Objetivo Funcional
La aplicación es un muro simple donde los usuarios pueden compartir fragmentos de código o saludos.
- **Entradas Requeridas (Inputs):**
  1. **Nombre:** (Texto) Quién lo envía.
  2. **Código:** (Texto/Snippet) Un código de referencia o ID.
  3. **Mensaje:** (Texto largo) El contenido o saludo.
- **Flujo:** El usuario llena los inputs -> Se guarda en Supabase -> Se actualiza la lista inferior en tiempo real.

## 2. Identidad Visual (Design System)
- **Tema:** ULTRA DARK.
- **Fondo:** El background debe ser negro absoluto (`#000000`) o gris muy oscuro (`#0a0a0a`).
- **Contraste:** Textos en blanco o gris claro (`#ededed`).
- **Estilo:** Minimalista, sin sombras pesadas. Bordes finos y sutiles.
